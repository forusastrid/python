파이썬 소개

파이썬은 높은 수준의 인터프리터 언어로, 다양한 분야에서 활용됩니다. 특히 웹 개발, 데이터 과학, 인공지능, 자동화 스크립트 등 여러 영역에서 사용됩니다. 파이썬의 가장 큰 특징은 가독성이 매우 뛰어나며, 문법이 간단하여 초보자도 쉽게 배우고 사용할 수 있다는 것입니다.

파이썬의 주요 특징

  • 간결하고 직관적인 문법
  • 강력한 표준 라이브러리 제공
  • 다양한 커뮤니티와 문서화된 자료들
  • 다양한 운영체제에서 사용 가능

참고: 파이썬은 Guido van Rossum에 의해 1991년에 처음 출시되었습니다.

변수와 자료형

파이썬에서 변수는 데이터를 저장하는 이름입니다. 파이썬은 동적 타이핑 언어로, 변수의 타입을 명시하지 않아도 됩니다. 변수에 저장된 데이터는 자료형에 따라 다르게 처리됩니다.

자료형 종류

  • 정수형 (Integer): 숫자, 예: 10, -5
  • 실수형 (Float): 소수점을 포함한 숫자, 예: 3.14, -0.001
  • 문자열 (String): 문자들의 집합, 예: "Hello", "Python"
  • 불린형 (Boolean): 참(True) 또는 거짓(False)

변수 예시

x = 10 # 정수형
y = 3.14 # 실수형
name = "파이썬" # 문자열
is_active = True # 불린형

참고: 파이썬에서 변수명은 문자, 숫자, 밑줄(_)을 사용할 수 있으며 숫자로 시작할 수 없습니다.

함수

함수는 재사용 가능한 코드 블록으로, 특정 작업을 수행하는데 사용됩니다. 함수를 정의할 때는 def 키워드를 사용하며, 입력값을 받아 결과를 반환할 수 있습니다.

함수 정의 예시

def greet(name):
    return "안녕하세요, " + name

print(greet("홍길동"))

위 코드에서는 greet라는 함수를 정의하여, 이름을 입력받고 인사말을 반환합니다.

참고: 파이썬에서는 함수의 매개변수에 기본값을 설정할 수도 있습니다.

반복문

반복문은 코드 블록을 반복해서 실행할 때 사용됩니다. 주로 사용되는 반복문은 for문과 while문입니다.

for문 예시

for i in range(5):
    print(i)

위의 예시에서는 range(5)를 사용하여 0부터 4까지의 값을 출력합니다.

while문 예시

i = 0
while i < 5:
    print(i)
    i += 1

참고: while문은 조건이 참인 동안 반복합니다.

조건문

조건문은 프로그램 흐름을 제어할 때 사용됩니다. 파이썬에서는 if, elif, else를 사용하여 조건을 분기할 수 있습니다.

조건문 예시

x = 10
if x > 0:
    print("x는 양수입니다.")
elif x == 0:
    print("x는 0입니다.")
else:
    print("x는 음수입니다.")

참고: 조건문은 여러 가지 조건을 분기할 때 유용하게 사용됩니다.

클래스와 객체지향 프로그래밍

파이썬에서는 객체지향 프로그래밍(OOP)을 지원하며, 이를 통해 클래스와 객체를 사용하여 프로그램을 구조적으로 설계할 수 있습니다.

클래스 정의 예시

class Dog:
    def __init__(self, name):
        self.name = name

    def bark(self):
        print(self.name + "가 짖습니다.")

위 코드는 Dog라는 클래스를 정의하고, 이름을 입력받아 짖는 메서드를 정의한 예시입니다.
